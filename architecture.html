Collector — stateless transforms

  • parseTopic() → structured metric
  • gear "N"→0, "1"-"6"→1-6, else drop
  • speed m/s × 3.6 → km/h
  • payload bare value or JSON envelope
  • event time from payload, fallback to receipt
  • unknown topics dropped at the edge

Writer — Resampler internals

  • Series sorted (ts, value) per field
  • LOCF valueAsOf(t) → last known ≤ t
  • fresh() null if older than 5 min
  • watermark maxTs − graceMs (10 s)
  • SoC Σ(socᵢ×capᵢ) / Σ(capᵢ)
  • final flush drops grace on shutdown

Postgres — guarantees

  • unique index (car_id, time)
  • upsert ON CONFLICT DO UPDATE
  • idempotent replay = same result
  • gap-free one row per 5 s always
  • NULL = stale or never observed
  • timestamp UTC, no timezone offset
